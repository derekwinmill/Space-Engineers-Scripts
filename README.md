# Space Engineers Scripts

This repository contains C# scripts for the **Space Engineers** programmable block.

## Repository purpose

- Store reusable in-game scripts
- Track script revisions in Git
- Document usage and setup notes

## Planned structure

- `scripts/` — production-ready programmable block scripts
- `examples/` — small focused examples/snippets
- `docs/` — setup notes, limitations, and usage instructions

## Getting started

1. Add a new script file under `scripts/`.
2. Paste script contents into a programmable block in Space Engineers.
3. Document any configuration/setup notes for that script.
